# assets/

This folder contains diagrams and technical documentation.

Suggested files:

```text
workflow_diagram.png      Screenshot or generated diagram of the LangGraph workflow
workflow_diagram.mmd      Mermaid source for the workflow diagram
technical_details.md      Detailed implementation notes
```

The root README keeps the project beginner-friendly. `technical_details.md` goes deeper into graph design, Azure deployment, and troubleshooting.
